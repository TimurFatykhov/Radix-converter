Radix converter iOS
==============
Calculate number from p1 to p2(<16) base radix

## Usage
Clone repositories.
```sh
$ git clone git@github.com:TimurFatykhov/Radix-converter.git
```
Install pods in Xcode project
```sh
pod install
```
Run app
